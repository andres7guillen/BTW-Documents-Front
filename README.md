# BTW-Documents-Front
UI for managing documents

Requisitos
Node.js (recomendado: ≥ 18)
npm (incluido con Node)

Instalación y ejecución:
# 1. Clonar el repositorio
git clone https://github.com/andres7guillen/BTW-Documents-Front.git
cd <ubicacion archivo donde se clonó>/btw-front-app

# 2. Instalar dependencias
npm install

# 3. Ejecutar la aplicación
npm start
